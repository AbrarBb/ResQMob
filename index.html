üöÄ How to Run the Project

Option 1: Use Node.js Version Manager (Recommended)

# Install nvm (Node Version Manager) # Windows: https://github.com/coreybutler/nvm-windows # macOS/Linux: curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash # Install and use Node.js 18 nvm install 18 nvm use 18 # Install dependencies npm install # Start the development server npm run dev

Option 2: Use Docker

# Create a Dockerfile FROM node:18-alpine WORKDIR /app COPY package*.json ./ RUN npm install COPY . . EXPOSE 8081 CMD ["npm", "run", "dev"]

Option 3: Manual Node.js Installation

Download and install Node.js 18.x from nodejs.org
